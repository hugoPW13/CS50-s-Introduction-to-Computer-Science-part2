# El Elfo Blanco' Website
#### Video Demo:  https://youtu.be/3Gjl2uKGwTM
#### Description:
El Elfo Blanco es un sitio web desarrollado utilizando HTML, CSS y Bootstrap 5, concebido como proyecto final del curso CS50X, que combina de manera integral elementos visuales, interactivos y de contenido con una temática navideña, enfocándose en brindar al usuario una experiencia envolvente y atractiva mediante la integración de colores festivos, imágenes pixeladas, tipografías y componentes visuales que evocan tanto la nostalgia de los estilos retro como la calidez y la alegría propias de la temporada, mientras que la estructura y la semántica del código buscan garantizar accesibilidad, buena organización y responsividad en todos los dispositivos. El archivo principal index.html define toda la arquitectura del sitio, comenzando con la declaración <!DOCTYPE html> y el elemento <html> con atributo lang="en" para establecer el idioma principal del documento, y en la sección <head> se incluyen etiquetas esenciales como <meta charset="UTF-8"> para la correcta codificación de caracteres, <meta name="viewport"> para asegurar que el sitio se adapte a diferentes tamaños de pantalla y dispositivos móviles, enlaces a Bootstrap 5 que proporcionan estilos predeterminados y componentes dinámicos, un archivo CSS externo llamado styles.css que contiene reglas personalizadas para garantizar que los colores, márgenes, tipografía, tamaños de imágenes y disposición de los elementos reflejen la identidad visual del proyecto, un favicon que refuerza la marca del sitio y un bloque <style> interno donde se configuran estilos específicos adicionales, como colores de fondo y texto, efectos hover y transiciones en imágenes y botones, disposición mediante flexbox para una alineación precisa y flexible de todos los elementos, así como estilos particulares para el header y el footer que aseguran una coherencia estética y jerarquía visual en todo el sitio. La estructura del <body> está organizada de manera lógica y jerárquica en distintas secciones, comenzando con el header, que utiliza las clases .box y .header-logo y contiene el logo del sitio, el título "El Elfo Blanco" y enlaces directos a GitHub y al curso CS50X, mientras que las imágenes y los iconos incluidos poseen efectos hover y escalado que aportan dinamismo y retroalimentación visual inmediata al interactuar con ellos, contribuyendo a una navegación más intuitiva y atractiva. La sección principal del sitio, definida con las clases .container y .content-wrapper, hace uso de flexbox para alinear de manera armoniosa las imágenes, bloques de texto y enlaces, e incluye tres bloques principales que representan el núcleo interactivo del sitio: una imagen central que al hacer clic activa la función sayThanks() de JavaScript, mostrando un mensaje emergente con felicitaciones navideñas y un agradecimiento al usuario, reforzando la sensación de interacción y participación; un enlace destacado a la sección de recetas que permite explorar contenido culinario navideño, mostrando el cuidado en la organización de la información y la integración de recursos externos; y un enlace al calendario de adviento, que brinda interacción diaria con sorpresas visuales y mensajes relacionados con la festividad, fomentando la exploración y el retorno de los usuarios al sitio, mientras que todas las imágenes están configuradas con la clase img-fluid para garantizar responsividad y poseen efectos de escala, cambio de color y transiciones suaves al pasar el cursor, mejorando tanto la estética como la experiencia de usuario. El footer, estructurado con las clases .bot y .bot2, contiene información de contacto, enlaces a redes sociales, objetivos del proyecto, créditos de desarrollo y detalles adicionales sobre la implementación del sitio, todo dispuesto con márgenes, padding y colores de fondo que armonizan con la paleta general del proyecto, mientras que las imágenes de redes sociales tienen tamaño controlado y enlaces directos, permitiendo conectividad externa sin comprometer la coherencia visual. Además, el sitio incluye un pequeño pero significativo script en JavaScript que ejecuta la función sayThanks(), generando un mensaje emergente cuando el usuario hace clic en la imagen principal, lo que añade un nivel de interactividad simple pero efectivo y crea un vínculo inmediato entre el usuario y el contenido, reforzando la experiencia global. La combinación de HTML semántico, CSS estilizado y JavaScript interactivo, junto con la integración de Bootstrap 5, no solo asegura que el sitio sea visualmente atractivo y consistente, sino también funcional y adaptable, ofreciendo una experiencia de navegación fluida, intuitiva y entretenida. En su conjunto, El Elfo Blanco representa un proyecto final completo y cuidadosamente elaborado que demuestra tanto los conocimientos adquiridos durante el curso CS50X como la creatividad aplicada a la presentación de un tema clásico y festivo como la Navidad, mostrando atención a la estética, la interactividad, la accesibilidad y la estructura del código, lo que convierte al sitio en un ejemplo destacado de cómo la programación web puede combinar funcionalidad, diseño y experiencia de usuario en un solo producto coherente y envolvente.
